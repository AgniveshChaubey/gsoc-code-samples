Bowtie is a meta-validator program that executes multiple JSON Schema validation implementations and collects the results. Currently there are supports available for 15 implementations across 10 programming languages and can run the official JSON Schema Test Suite, providing a single place for users and implementers to see the results across many implementations. This enables an easy way to compare how closely an implementation follows the specification, providing transparency and leading to improvement in the community. Bowtie also provides a uniform interface to the supported implementations, allowing users to quickly access results without needing to learn each implementation's language-specific API. Bowtie is a useful tool for anyone working with JSON data and can help with schema and instance validation across multiple implementations.
